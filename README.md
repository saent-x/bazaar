# Tiv Live Voice Assistant

An open, community-oriented project to make natural live AI conversation
available in Tiv. The intended experience is a continuous voice session with
automatic turn-taking, spoken Tiv responses, conversation memory, and the
ability to interrupt the assistant.

The service is intended to remain free and be sustained by donations. The
project is currently in the feasibility and MVP-definition stage; no application
stack has been initialized yet.

## Current objective

Build a voice laboratory that lets native Tiv speakers hold short, low-latency
conversations while the team measures recognition quality, response quality,
speech naturalness, interruption behavior, and latency. The evidence from this
prototype will determine which components need adaptation or additional data.
